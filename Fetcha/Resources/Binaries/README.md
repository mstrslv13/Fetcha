# Binary Dependencies

This directory should contain the following binaries:
- `yt-dlp` - The video downloader
- `ffmpeg` - For video/audio processing

## Installation

The app will automatically detect if you have these installed via Homebrew.

To install manually:

```bash
brew install yt-dlp ffmpeg
```

Or download directly:
- yt-dlp: https://github.com/yt-dlp/yt-dlp/releases
- ffmpeg: https://ffmpeg.org/download.html

Place the binaries in this directory if you want to bundle them with the app.
